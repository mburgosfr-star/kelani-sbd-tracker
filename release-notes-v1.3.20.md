# Kelani SBD Tracker v1.3.20

- Smart Training now balances heavy Squat, Bench Press, and Deadlift sessions more effectively across consecutive workouts.
- The same lift is no longer selected as the heavy primary lift on consecutive training days when another suitable lift is available.
- Light secondary work is now weighted differently from heavy primary work when balancing training frequency.
- The next Smart workout is refreshed immediately after completing a workout.
- Meet readiness now uses successful performances from the current cycle and realistic opening attempts.
- Accessory action buttons are now aligned correctly beneath their set columns.
