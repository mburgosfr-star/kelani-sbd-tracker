# Kelani video / YouTube structure

Purpose: keep exercise guidance simple, practical, and app-focused.

## Initial scope

Kelani should start with one short execution video per main lift:

1. Squat
2. Bench Press
3. Deadlift

Each video should support the in-app exercise guide modal. The goal is not a large content library, but one clear reference video per main lift.

## Video format

Each video should be short and direct:

- Setup
- Main execution cues
- Common safety points
- What a clean rep should look like

## In-app use

The Workout screen exercise guide modal can link to one video per main lift:

- Squat guide → Squat video
- Bench Press guide → Bench Press video
- Deadlift guide → Deadlift video

No long documentation and no large list of random exercise videos.

## Later optional videos

After the main lift videos exist, Kelani can optionally add short videos for replacement exercises:

- Belt Squat
- Chest Press
- Posterior Chain / Deadlift Alternative

These are optional and should not block app releases.
