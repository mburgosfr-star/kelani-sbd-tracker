# Kelani SBD Tracker v1.1 plan

v1.1 should make Kelani feel more guided without making workout execution more complex.

## Goal

Make Kelani clearer before and after training, while keeping the workout screen fast and simple.

## Priority 1: onboarding

Improve the first-run experience so new users understand:

- what Kelani is
- who it is for
- how maxes are used
- how the program works
- what to expect from a cycle

## Priority 2: program explanation

Add simple explanations around the program structure:

- top sets
- back-off sets
- lighter/heavier weeks
- progression across a cycle
- meet preparation

## Priority 3: cycle review

At the end of a cycle, summarize useful progress:

- completed workouts
- lift changes
- total change
- failed or adjusted sets
- bodyweight trend if available
- simple next-cycle message

## Priority 4: feedback loop

Improve feedback without tracking:

- clearer feedback option
- optional email feedback
- GitHub issue option for technical users
- light prompt after completing a cycle

## Not for v1.1

Avoid for now:

- custom program builder
- accounts
- analytics
- paid features
- major workout screen redesign
- new program paths
