# Kelani SBD Tracker 1.3.22

This release improves Smart workout variety and set layout.

## Improvements

- Uses three work sets per lift on mixed-lift Smart training days, keeping set grids complete.
- Avoids repeating the same recent primary-lift prescription when safe variation is available.
- Preserves back-off volume while varying the primary top stimulus.
- Refreshes pending Smart workouts when prescription logic changes.

## Reliability

- Added regression coverage for mixed-day set counts and repeated primary-lift prescriptions.
