# Kelani SBD Tracker

Kelani SBD Tracker is a simple powerlifting app to track Squat, Bench and Deadlift training cycles.

## Features

- Structured SBD training programs  
- Automatic progression based on performance  
- Rest timer with audio signals  
- Bodyweight tracking  
- Detailed statistics and graphs  
- Offline-first (no internet required)  
- Multilingual interface
