# Kelani SBD Tracker 1.3.23

This release completes Smart workout set grids on single-lift training days.

## Improvements

- Uses five work sets on standard single-lift Smart training days, preventing incomplete set grids.
- Keeps the existing six-by-six Bench prescription when that higher-volume option is appropriate.
- Refreshes pending Smart workouts through Smart prescription version 8.

## Reliability

- Added regression coverage for single-lift set counts.
- Preserved mixed-lift three-set grids and existing Smart progression behavior.
